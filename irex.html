About

Irex is an free, open-source, fun and smart virtual assistant developed by HistorVical which is primarily available for Windows.

Download

Download the assistant by clicking the button below. Please note that python is required to run the python file and by clicking the button, you will be redirected to a google drive link. You can proceed to download the file there.

Download Python File

Commands

Hello - Gives a warm and randomized hello message.
What is the time? - Shows time and date.
What is the date? - Shows time and date.
Tell me a joke - Tells a joke.
Search Google - Searches a specific topic on google.
Quit or Exit - Quits the program after confirmation
